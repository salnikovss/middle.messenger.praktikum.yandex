# YP Messenger - учебный проект мессенджера

# Команды

## Разработка проекта

```bash
npm install
npm run dev
```

## Сборка проекта

```bash
npm install
npm run build

```

## Запуск проекта

```bash
npm install
npm run start
```

## Статический анализ кода и стилей


```bash
npm run lint
npm run lint:fix
npm run stylelint
npm run stylelint:fix
```
